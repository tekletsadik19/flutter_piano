# Ruth — App Overview

## What is Ruth?

Ruth is a cross-platform Flutter piano learning app. It combines an interactive MIDI piano with a structured learning experience, allowing users to learn piano from scratch through lessons, exercises, and songs — all within a single app.

## Vision

> A complete piano learning companion. Play, learn, and progress — on any device.

## Core Concept

The app has two sides:
1. **Learn** — structured lessons, exercises, theory, and songs that guide the user from beginner to advanced
2. **Play** — a full interactive MIDI piano keyboard for free play and guided practice

These two sides are connected: lessons open the piano in a guided mode where the app listens, highlights keys, and gives feedback.

## Target Users

- Complete beginners with no prior piano knowledge
- Self-taught players who want structured guidance
- Intermediate players looking to improve technique and theory
- Anyone wanting to learn a specific song

## Platforms

- iOS
- Android
- macOS
- Windows
- Web

## Tech Stack

- **Flutter** (Dart) — cross-platform UI
- **Riverpod** — state management
- **audioplayers** — MIDI/audio playback
- **google_fonts** — Outfit font
- **shared_preferences** — local settings persistence
